Description

A website I personally use on a daily basis, QRCodeGen is a lightweight generator that turns any text or URL into a scannable QR code in seconds. Built to be fast, responsive, and accessible, it fits seamlessly into projects that need simple sharing or contact-less experiences.

This project demonstrates intuitive controls, live previews, and mobile-friendly layouts so users can create clean QR codes on any device without extra downloads.
